# Rock Paper Scissors Game

## Overview

This is a Python-based Rock Paper Scissors game where the user competes against the computer. The game includes score tracking, multiple rounds, winner determination, and a user-friendly interface.

## Features

* User vs Computer gameplay
* Random computer selection
* Multiple rounds
* Score tracking
* Input validation
* Game summary at the end

## Technologies Used

* Python 3
* Random Module

## Run the Project

```bash
python rock_paper_scissors.py
```
