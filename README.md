# 🎮 Rock Paper Scissors Pro

## 📌 Overview

Rock Paper Scissors Pro is a modern desktop game developed using Python and Tkinter. The application provides an interactive graphical user interface where players can compete against the computer in the classic Rock-Paper-Scissors game.

The project includes score tracking, game result display, reset functionality, and a modern dark-themed user interface designed to enhance the user experience.

---

## ✨ Features

* Interactive Graphical User Interface (GUI)
* User vs Computer Gameplay
* Random Computer Selection
* Real-Time Score Tracking
* Round Result Display
* Reset Game Functionality
* Exit Button
* Modern Dark Theme Design
* User-Friendly Interface

---

## 🛠 Technologies Used

* Python 3
* Tkinter (GUI Library)
* Random Module

## ▶️ How to Run

### Step 1

Install Python 3 on your system.

### Step 2

Open Command Prompt or Terminal.

### Step 3

Navigate to the project folder.

```bash
cd Rock-Paper-Scissors-Pro
```

### Step 4

Run the application.

```bash
python rock_paper_scissors.py
```
